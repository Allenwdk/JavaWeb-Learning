# Java 17 新特性

## 学习目标

- 了解 Java 版本演进历史
- 掌握 Java 17 中适合 Web 开发的新特性
- 理解 record、switch 表达式、文本块等特性

## Java 版本历史

Java 从 2004 年发布 Java 5 到现在，经历了多次重大更新：

| 版本 | 发布时间 | 重大特性 |
|------|----------|----------|
| Java 5 | 2004 | 泛型、注解、枚举 |
| Java 8 | 2014 | Lambda、Stream API、Optional |
| Java 11 | 2018 | 局部变量类型推断 var |
| Java 17 | 2021 | record、switch 表达式、文本块 |
| Java 21 | 2023 | 虚拟线程（Project Loom） |

**为什么选 Java 17？**
- Java 17 是 LTS（长期支持）版本，会收到官方支持到 2029 年
- 包含了 Java 11 和 Java 17 之间的重要特性
- Spring Boot 3.x 最低要求 Java 17

## 检查你的 Java 版本

在终端执行：

```bash
java -version
```

如果版本低于 17，需要安装 JDK 17 或更高版本。推荐安装 **Temurin**（Eclipse 维护的发行版）或 **Corretto**（Amazon 维护的发行版）。

## Java 17 核心特性

### 1. Record 记录类

当你只需要存储数据、不需要方法时，用 record 代替 class：

```java
// 传统方式 — 需要 30+ 行代码
public class User {
    private final String name;
    private final int age;
    
    public User(String name, int age) {
        this.name = name;
        this.age = age;
    }
    
    public String getName() { return name; }
    public int getAge() { return age; }
    
    @Override
    public boolean equals(Object o) {
        if (this == o) return true;
        if (!(o instanceof User)) return false;
        User user = (User) o;
        return age == user.age && Objects.equals(name, user.name);
    }
    
    @Override
    public int hashCode() {
        return Objects.hash(name, age);
    }
    
    @Override
    public String toString() {
        return "User{name='" + name + "', age=" + age + "}";
    }
}

// Record 方式 — 只需 1 行
public record User(String name, int age) {}
```

record 自动提供：
- 构造器
- getter 方法（ getName()、getAge() ）
- equals()、hashCode()
- toString()

**在 Web 开发中的应用：** record 非常适合做 DTO（数据传输对象）：

```java
public record UserDTO(String name, int age, String email) {}
public record ApiResponse<T>(int code, String message, T data) {}
```

### 2. Switch 表达式

switch 不再只是语句，它可以是表达式（有返回值）：

```java
// 旧方式
String result;
switch (day) {
    case "MON":
        result = "星期一";
        break;
    case "TUE":
        result = "星期二";
        break;
    default:
        result = "未知";
}

// Java 17 方式
String result = switch (day) {
    case "MON" -> "星期一";
    case "TUE" -> "星期二";
    case "WED" -> "星期三";
    default -> "未知";
};
```

**箭头语法（`->`）的优势：**
- 不需要 `break` — 不会穿透
- 代码更简洁
- 可以返回一个值

**多分支情况：**

```java
String type = switch (status) {
    case 200, 201 -> "成功";
    case 400, 401, 403 -> "客户端错误";
    case 404 -> "未找到";
    case 500, 502, 503 -> "服务器错误";
    default -> "未知状态";
};
```

**在 Web 开发中的应用：** 处理 HTTP 状态码、订单状态、用户角色等：

```java
public String getStatusText(int statusCode) {
    return switch (statusCode) {
        case 200 -> "OK";
        case 201 -> "Created";
        case 400 -> "Bad Request";
        case 404 -> "Not Found";
        case 500 -> "Internal Server Error";
        default -> "Unknown";
    };
}
```

### 3. 文本块（Text Blocks）

用 `"""` 定义多行字符串，不用写 `\n`：

```java
// 旧方式 — 需要拼接
String sql = "SELECT id, name, email " +
             "FROM users " +
             "WHERE age > ? " +
             "ORDER BY name";

// 文本块方式
String sql = """
    SELECT id, name, email
    FROM users
    WHERE age > ?
    ORDER BY name
    """;

// HTML 内容
String html = """
    <html>
      <body>
        <h1>Hello</h1>
        <p>Welcome to our site!</p>
      </body>
    </html>
    """;
```

**在 Web 开发中的应用：** 在 MyBatis 中写 SQL、生成 HTML 响应、处理 JSON 模板等：

```java
String json = """
    {
        "name": "张三",
        "age": 20,
        "email": "zhangsan@example.com"
    }
    """;
```

### 4. 模式匹配 instanceof

```java
// 旧方式
if (obj instanceof String) {
    String s = (String) obj;  // 需要手动强转
    System.out.println(s.length());
}

// Java 17 方式
if (obj instanceof String s) {
    System.out.println(s.length());  // s 已经是 String 类型
}
```

### 5. 集合工厂方法

```java
// 旧方式
List<String> fruits = new ArrayList<>();
fruits.add("apple");
fruits.add("banana");
fruits.add("orange");

Map<String, Integer> ages = new HashMap<>();
ages.put("张三", 20);
ages.put("李四", 22);

// Java 17 方式
List<String> fruits = List.of("apple", "banana", "orange");
Map<String, Integer> ages = Map.of("张三", 20, "李四", 22);

// 注意：List.of 和 Map.of 创建的是不可变集合
// 如果需要修改，用 List.copyOf() 后再转
List<String> mutable = new ArrayList<>(List.of("apple", "banana"));
mutable.add("orange");
```

## 练习任务

### 任务1：用 record 定义数据类

在你的 Spring Boot 项目中创建一个 UserDTO 类：

```java
public record UserDTO(String name, int age, String email) {}
```

然后在 Controller 中返回它：

```java
@RestController
public class UserController {
    @GetMapping("/user")
    public UserDTO getUser() {
        return new UserDTO("张三", 20, "zhangsan@example.com");
    }
}
```

运行项目，访问 http://localhost:8080/user，观察返回的 JSON。

```java
@RestController
public class StatusController {
    @GetMapping("/status/{code}")
    public String getStatus(@PathVariable int code) {
        return switch (code) {
            case 200 -> "OK";
            case 404 -> "Not Found";
            case 500 -> "Internal Server Error";
            default -> "Unknown";
        };
    }
}
```
